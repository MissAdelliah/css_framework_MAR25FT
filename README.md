Comit is a SoMe platform, for sharing work between developers. For inspiration, information or to give/resieve help on working projects. This to connect developers.

## Build with

/feed/index.html - Feed page with posts, search, sorting, filters and create post form

## Development 
Run Sass in watch mode:
npm run dev

This watches the Sass files and compiles them into:
styles/css/style.css

Run the production build:
npm run build

-made sure Bootstrap and Sass was updated 

The project uses Bootstrap through npm and does not use Bootstrap CDN links.
